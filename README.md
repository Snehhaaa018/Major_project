# how to run 

clone the repo

```
https://github.com/Snehhaaa018/Major_project.git
```

- make a virtual envirment by 
## on windows

```
python -m venv venv 
```

- activate the envirment on windows 
```
venv/Scripts/activate
``` 

## on linux and mac

```
python3 -m venv venv 
```

- activate the envirment on windows 
```
source venv/bin/activate
``` 


## install requirements by

```
pip install -r requirements.txt
```

- run the server
```
python manage.py runserver
```
